# tcuida_dwes_project

Proyecto realizado para la asignatura de **Desarrollo Web en Entorno Servidor (DWES)** durante el primer trimestre.

## Descripción

Este proyecto es una aplicación web desarrollada principalmente en **PHP**, con apoyo de **CSS** y **JavaScript**, que simula la gestión de una plataforma llamada **Tcuida**. El objetivo es practicar y demostrar los conocimientos adquiridos en el desarrollo de aplicaciones web del lado del servidor.

## Características principales

- Gestión de usuarios (registro, login, roles, etc.)
- Interfaz de usuario con estilos personalizados en CSS
- Funcionalidades dinámicas mediante JavaScript
- Acceso y gestión de datos mediante PHP y MySQL

## Estructura del proyecto

```
/
├── css/              # Hojas de estilo
├── js/               # Scripts JavaScript
├── includes/         # Archivos PHP reutilizables (conexión BD, cabeceras, etc.)
├── main.php          # Página principal de la aplicación
├── ...
```

## Instalación y uso

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/rxy94/tcuida_dwes_project.git
   ```
2. **Configura la base de datos:**
   - Importa el script SQL proporcionado (si existe) en tu servidor MySQL.
   - Ajusta los parámetros de conexión en el archivo correspondiente (por ejemplo, `includes/conexion.php`).

3. **Levanta el entorno local:**
   - Puedes usar XAMPP, WAMP o cualquier servidor compatible con PHP y MySQL.
   - Coloca el proyecto en la carpeta `htdocs` (o equivalente) y accede desde tu navegador.

## Tecnologías utilizadas

- **PHP** 
- **CSS** 
- **JavaScript**
- **MySQL**
